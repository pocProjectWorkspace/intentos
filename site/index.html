What It Does

Language is the interface

Tell IntentOS what you need in plain English. It routes your intent to specialized agents that execute on your machine.

🔍
File Intelligence
Find, organize, rename, and manage files using natural language. "Show me my largest files" just works.
🌐
Web Research
Search the web, fetch pages, extract data. Results stay on your device. No tracking, no history leaks.
📄
Document Creation
Create, edit, and convert documents. Word, PDF, and more — all processed locally with zero cloud uploads.

Semantic Memory

Privacy & Security

The file never leaves

Built for the device that holds your most sensitive documents. Enterprise-grade security is the default, not an upgrade.

Local-First AI
AI runs on your hardware via Ollama. File contents, paths, and embeddings never leave your machine — not even in cloud mode.
Files processed locally
Works fully offline
No telemetry
Encrypted Credentials
AES-256-GCM encryption with HKDF-SHA256 key derivation. API keys stored in your OS keychain — never in config files.
macOS Keychain integration
Windows Credential Manager
No .env files in production
Sandboxed Agents
Every agent runs in a permission sandbox. Read-only, workspace-write, or full-access — with destructive actions requiring your confirmation.
Per-agent permission policies
Path grant enforcement
Destructive action confirmation
Leak Detection
Every output is scanned for secrets before it reaches any AI model. API keys, PEM files, bearer tokens — blocked, redacted, or flagged.
Input & output scanning
Prompt injection defense
Append-only audit log
